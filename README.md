## Install Instructions

- You will need docker and docker-compose to run the application.
- To start the application in development mode run the command

    ```
    npm run up
    ```
    or the alias 
    ```
    make up
    ```
- Run tests with
    ```
    npm run test
    ```
    or the alias
    ```
    make test
    ```
- Production Deployment
    ```
    npm run up:prod
    ```
    or the alias
    ```
    make prod
    ```
